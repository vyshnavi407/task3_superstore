# task3_superstore

# Project Overview

This project focuses on analyzing the Superstore Sales dataset and developing an interactive sales dashboard using Python, SQL, and Jupyter Notebook. The dashboard provides key business insights through visualizations and KPI metrics such as Total Sales, Total Profit, and Total Orders.

# Objectives

* Perform sales data analysis.
* Execute SQL queries for business insights.
* Create meaningful visualizations.
* Develop a colorful dashboard using HTML and Python.
* Present key performance indicators (KPIs).

## Tools & Technologies

* Git & GitHub
